CheatSheets do Ensino Médio

O que é?

Este é um projeto de código aberto que tem como intenção listar da maneira mais simples possível a maior quantidade de fórmulas essenciais para Ensino Médio (principalmente para o vestibular).

Como é feito?

O projeto foi construído com HTML, CSS e Javascript. O código foi construído com base em HTML5Boilerplate e Pure CSS.

Como contribuir?

Para contribuir, envie um email para victorsenam@gmail.com ou acesse o projeto no GitHub.

Durante o resumo, haverão indicações de áreas onde não foram feitas contribuições para o projeto.
